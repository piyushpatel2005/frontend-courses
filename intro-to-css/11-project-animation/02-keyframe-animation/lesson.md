# Keyframe Animation

Add a looping keyframe animation to the badge element.

## Step-by-step instructions

1. Define `@keyframes pulse` with scale transform steps.
2. Apply `animation: pulse 1.5s infinite;` to `.badge`.
3. Verify both animation and hover transition are visible in the page.
