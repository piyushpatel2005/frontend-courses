# Project: Create Animated Components

Build a button and notification badge with transitions and keyframe animation.

## Step-by-step instructions

1. Add a `.cta-btn` button and a `.badge` element.
2. Add `.cta-btn { transition: transform 0.2s ease, background-color 0.2s ease; }`.
3. Add `.cta-btn:hover { transform: translateY(-2px); }`.
4. Define `@keyframes pulse` animation.
5. Apply `animation: pulse 1.5s infinite;` to `.badge`.
6. Ensure animation and hover are both visible in the page.

## Goal

Practice modern CSS interaction patterns with smooth motion.
