# CSS Selectors

Selectors choose which HTML elements should receive a style rule.

## Exercise

1. Style all `p` elements with `line-height: 1.6;`.
2. Style `.note` with `background-color: #fff4cc;`.
3. Style `#hero-title` with `color: #7a1fa2;`.
