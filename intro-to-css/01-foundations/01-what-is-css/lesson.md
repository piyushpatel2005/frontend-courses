# What is CSS?

CSS controls how HTML elements look. It can style text, spacing, colors, and layout.

## Exercise

1. Add a `<style>` tag inside `<head>`.
2. Style `body` with `background-color: #f6f8fb;`.
3. Style `h1` with `color: #1b4d8c;`.
