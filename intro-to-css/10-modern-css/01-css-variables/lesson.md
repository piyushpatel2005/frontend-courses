# CSS Variables

CSS custom properties make design tokens reusable and consistent.

## Exercise

1. In `:root`, define `--brand: #0ea5e9;`.
2. Use `var(--brand)` for `.title` color.
3. Define `--space: 16px;` and use it for `.panel` padding.
