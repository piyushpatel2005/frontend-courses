# Transitions

Transitions animate style changes between states.

## Exercise

1. Add `transition: transform 0.2s ease;` on `.btn`.
2. Add `.btn:hover { transform: translateY(-2px); }`.
3. Add `transition: background-color 0.2s ease;` on `.btn`.
