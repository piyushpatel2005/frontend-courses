# Project: Create a Resume with Better Styles

Design a polished resume page using typography, layout, and responsive techniques.

## Step-by-step instructions

1. Add a `.resume` wrapper with sections: header, summary, skills, experience, and contact.
2. Define variables in `:root` (for colors and spacing).
3. Style header with clear hierarchy using font size, weight, and spacing.
4. Use grid for two-column layout on desktop (`grid-template-columns: 2fr 1fr`).
5. Add card-style sections with border, padding, and subtle background color.
6. Add responsive media query at `max-width: 768px` to collapse to one column.
7. Add links for email/portfolio with hover state.

## Goal

Deliver a clean, professional, responsive resume design.
