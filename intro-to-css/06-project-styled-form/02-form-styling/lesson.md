# Form Styling

Add layout, spacing, and focus feedback styles to your contact form.

## Step-by-step instructions

1. Set `.contact-form` to `display: grid;` and `gap: 12px;`.
2. Style text fields and textarea with `padding`, `border`, and `border-radius`.
3. Add `input:focus, textarea:focus, select:focus` styles with visible outline or border color.
