# Project: Create a Styled Form

Build a form UI with readable spacing and clear input focus feedback.

## Step-by-step instructions

1. Create a `<form class="contact-form">` with name, email, topic select, and message textarea.
2. Set `.contact-form` to `display: grid;` and `gap: 12px;`.
3. Style text fields and textarea with `padding`, `border`, and `border-radius`.
4. Add `input:focus, textarea:focus, select:focus` styles with visible outline or border color.
5. Add a submit button with custom background color and hover effect.
6. Keep labels above controls for readability.

## Goal

Users should clearly see which field is active and submit the form comfortably.
