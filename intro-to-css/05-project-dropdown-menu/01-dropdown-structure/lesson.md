# Dropdown Structure

Set up the HTML structure and positioning context for a CSS-only dropdown menu.

## Step-by-step instructions

1. Add a wrapper `.dropdown` with a button `.dropdown-toggle` and a panel `.dropdown-menu`.
2. Add at least 3 links inside `.dropdown-menu`.
3. Set `.dropdown` to `position: relative;`.
