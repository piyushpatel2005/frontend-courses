# Responsive Design

Responsive design adapts layouts to different screen sizes.

## Exercise

1. Set `.layout` to `display: grid;`.
2. Set default `grid-template-columns: 1fr;`.
3. Add media query `@media (min-width: 768px)` and set `grid-template-columns: 1fr 1fr;`.
