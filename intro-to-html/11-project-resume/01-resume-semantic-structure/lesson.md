# Project 3: Create a Resume (Semantic Structure)

Build a one-page HTML resume using semantic tags.

## Requirements

1. Add `<header>` with your name and professional title.
2. Add `<main>` with at least 3 `<section>` blocks:
   - Professional Summary
   - Skills
   - Work Experience
3. Add one unordered list in Skills.
4. Add `<footer>` with contact information.
