# Project 3: Resume (Links, Media, and Actions)

Enhance your resume page with actionable links and media.

## Requirements

1. Add a `<figure>` with profile image and `<figcaption>`.
2. Add at least 2 external links (portfolio, GitHub, LinkedIn).
3. Add one `mailto:` link for contact action.
4. Add one downloadable resume link using `download` attribute.
