Why Semantic HTML Matters

Published on March 10, 2024

Semantic HTML gives meaning to the structure of a web page. Elements like <article>, <nav>, and <footer> tell browsers and assistive technologies exactly what each part of the page is for.

When screen readers encounter a <nav> landmark, they can offer users a shortcut to jump straight to navigation. This dramatically improves the experience for people who use assistive technology.
